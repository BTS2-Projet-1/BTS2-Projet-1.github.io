Le constructeur initalise et ouvre la base de données automatiquement grâce aux attibuts de la classe. Si l\textquotesingle{}ouverture de la base de données est correcte un attibut \char`\"{}m\+\_\+\+Data\+Base\+Is\+Ready\char`\"{} accessible avec \hyperlink{class_c_s_q_lite___local___d_b_ad3380dd61189863c1d2d5e681b853e0e}{Get\+\_\+\+Data\+Base\+Is\+Ready()} qui revoie avec un booléen, si la connexion avec la base de données est effectuée sans erreurs vrai (true) ou avec un echec faux (false).

Dans le cas ou le programme tente de réaliser des actions avec la base de données avec l\textquotesingle{}attibut \char`\"{}m\+\_\+\+Data\+Base\+Is\+Ready\char`\"{} égal à faux (false), les fonctions membres retouneront une erreur et aucunes actions demendées seront réalisées.
