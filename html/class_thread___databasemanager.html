Constructeur de la classe Thread_Databasemanager, qui hérite de QThread pour la gestion de d'un thread permettant la gestion de la base de données, sans bloquer l'IHM.

Initialise les différents attributs et connecte les signaux.

Paramètres
ConsoleWindows*TConsoleWindows, SQLite_Local_DatabaseManager *DatabaseManager, CSQLite_Local_DB *BD.
*TConsoleWindows pointeur valide sur la classe parent ConsoleWindows
*DatabaseManager pointeur valide sur une instance de SQLite_Local_DatabaseManager pour faire remonter les informations par les signaux
*BD, pointeur valide sur une instance de CSQLite_Local_DB pour la communication avec la base de données.
